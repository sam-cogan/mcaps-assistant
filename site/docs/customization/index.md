# Customization

This repo is designed to be forked and tailored. The `.github/` directory is where all of Copilot's behavior is defined — in plain Markdown files you can edit directly.

!!! tip "No code required"
    Think of `.github/` as your team's operating manual for Copilot. Every file shapes what Copilot knows, how it reasons, and what it says. Edit freely — you can't break CRM by editing a Markdown file.
