# Prompts by Role

Copy-paste any of these into the Copilot chat window. Each prompt triggers one or more skills automatically — you don't need to name the skills, just describe what you need.

!!! tip "New here?"
    Start with `/getting-started` or `/my-role` instead. Come back to these once you're comfortable.

---

## Getting Oriented

These work for any role:

| What You Want | Prompt |
|---------------|--------|
| Check your CRM identity | `Who am I in MSX?` |
| See your active pipeline | `Show me my active opportunities.` |
| Understand available tools | `What MCP tools do I have available for MSX?` |
